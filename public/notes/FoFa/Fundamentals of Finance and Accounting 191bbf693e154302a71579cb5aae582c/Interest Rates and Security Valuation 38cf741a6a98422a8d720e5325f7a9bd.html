Various Interest Rate Measures

Coupon Rate

  • Periodic cash flow a bond issuer contractually promises to pay a bond holder
  • Fraud → You can borrow your company shares for 4000/- and sell them, then make their value go down, buy for 1000/- and then give it back at 4000 🙂 → Profit of 3000
  • Demat Accounts → Dematerialized Accounts
  • In simple words, it’s basically interest rate but named coupon rate
  • Bond is a debt while share is the ownership right

  • This is the interest rate someone should receive to cover up for their money

Expected Rate of Return

  • Rates participants would earn by buying securities at current market prices
  • If E(r) ≥ r or Current Market Price ≤ Present Value you should buy this security else no
  • We have two ex ante terms by now, one is the expected rate of return and other is the required rate of return
  • The req rate is used to calculate a fair present value of a financial security while expected rate is a discount rate used in conjunction with current market price

Realized Rate of Return

  • Rates actually earned on investments
  • This is ex post and not ex ante

Yield to Maturity

  • Yield to Maturity or ytmm is defined as the return the bond holder will earn on the bond if he/she buys the bond at the current market price, receives all coupon and principal amounts and holds the bond until it matures

Duration Analysis

Impact of Interest Rate changes on Security Values

  • When required rate of return increases from 8 to 10%, the present value decreases
  • Let’s assume coupon rate is 10%, if the required rate is just 8%, then the value of the bond is higher than what we expect in face value

Impact of Maturity on Security Values

  • A bond’s price sensitivity is measured by the percentage change in it’s present value for a given change in interest rates
  • The shorter the time remaining to maturity, the closer a bond’s price is to it’s face value
  • The further a bond is from maturity, the more sensitive the price is it’s face value
  • The intuition behind this is that nobody would pay much more than the face value of the bond and any remaining (in this case semiannual) coupon payments just prior to maturity since these are the only cash flows left to be paid on the bond.
  • Thus, the time value effect is reduced as the maturity of the bond approaches.

Impact of Coupon Value on Security Values

  • Higher the coupon rate, higher the present value of the bond

Duration

  • If you have a 1 billion $ bond, if the required rate of return changes by 0.25%
    • Face Value is 1 billion
    • Coupon is 10%
    • Discount Rate is 8%
    • Interest rate increases to 8.25% Or increase to 8.75%
  • All bond issuers are going to cry a lot
  • Interest Rates are gonna change in small portions and we use duration analysis to find a relation between maturity, coupon rate etc.
  • 100bps = 1% change where bps means basis points
  • We use bps because humans work on a large scale 🙂
  • Duration is the weighted average time to maturity on a financial security using the relative present values of cash flows as weights
  • Duration also has economic meaning as the sensitivity or elasticity of that asset or liability’s value to small interest rate changes → either required rate or return or yield to maturity
  • For a zero bond, duration = time of the bond because you are not getting any coupons in between

Features of Duration

  • Higher the coupon rate, shorter the duration, because you recover most money before only
  • As rate of return increases, the duration decreases, because if you are getting more in return that would mean you would recover earlier → Because of low present value
  • As maturity period increases, the duration also increases at a decreasing rate. We see that as maturity period increases, the number of coupons increases, so the duration also increases due to more spread out coupons
