Costs in short run

Total Cost = Total Fixed Costs + Total Variable Costs

Fixed Costs

  • These costs cannot be avoided in the short run but in long run the firm may decide to leave the market

Total Fixed Cost

  • Sometimes called the overhead
  • These do not change with output even if output is 0

Average Fixed Cost

  • This is the total fixed cost divided by the number of units of outputs
  • Average Fixed Cost reduces as number of units of outputs increases
  • This is because the same fixed cost is being spread over a larger range which leads to spreading overhead
  • Average fixed cost declines as quantity rises

Variable Costs

Total Variable Costs

  • Sum of those costs that vary with the level of output in the short run
  • To produce more output, we need give the firm more input
  • Notice the firm’s fixed costs do not come into the decision process: those costs stay the same no matter what the firm does and so can be put to the side when making a decision

Total Variable Cost Curve

  • Graph that shows the relationship between total variable cost and the level of a firm’s output
  • At any given level of output, TVC depends on
    • techniques of production that are available
    • the prices of input that are required by each technology

Marginal Costs and Average Costs

  • Increase in total cost of production that results from producing 1 more unit of output
  • Therefore MC = d(TVC)/dq = d(TVC)
  • Also AC = TVC/q so if we are given a curve Q^3-20Q^2-240Q, then AC = Q^2 - 20Q and when we differentiate, we get 2Q-20 = 0, Q = 10
  • Marginal cost measures the additional cost of inputs required to produce each successive unit of output.
  • If production function is given and you need to find marginal cost, differentiate cost function

Shape of Marginal Cost Curve in the Short Run

  • The assumption of a fixed run in the factors of production in the short run means that the firm is stuck at it’s current max scale
  • If each additional unit of labor adds less and less to total output, but input prices remain the same, it follows that each additional unit of output costs more to produce. In other words, diminishing returns, or decreasing marginal product, imply increasing marginal cost
  • Notice that up to q, marginal cost decreases and the variable cost curve becomes flatter. The slope of the total variable cost curve is declining—that is, total variable cost increases, but at a decreasing rate. Beyond q units of output, marginal cost increases and the total variable cost curve gets steeper—total variable costs continue to increase, but at an increasing rate
  • Intersection of MC and AVC is the minimum cost and also if MC < AVC AVC decreases and if MC > AVC, AVC increases
  • If the average variable cost curve is above the marginal cost curve, then marginal costs can either be increasing or decreasing
  • If marginal cost curve is below the average variable cost curve, then average variable cost is decreasing

Output Decisions

  • If a firm cannot sell it’s product for more than the cost of production, it will not be in the business long

Perfect Competition

  • This means we have relatively small firms producing identical or homogenous products
  • In a perfectly competitive industry, no single firm has any control over prices.
  • Firm faces a perfectly elastic demand curve for it’s individual product

Total Revenue and Marginal Revenue

  • TR = p * q
  • Marginal revenue (MR) is the added revenue that a firm takes in when it increases output by 1 additional unit.
  • If a firm producing 100 units of output per month increases that output to 101 units per month, that added unit will increase the firm’s revenue.
  • If the market price for this product is $5.00 per unit, then the increase in revenue from one unit, or the marginal revenue, will be $5.00.
  • For a competitive firm, marginal revenue is equal to the current market price of each additional unit sold.

Comparing Costs and Revenues to Maximize Profit

  • Assumptions
    • Perfectly Competitive Industry
    • Firms want to Maximize Profit
  • When q = 100, MC = 2.5 and MR = 5 we go on so that we increase profit
  • When q = 250, MC = 4 and MR = 5 so we move further
  • When q = 340, MC = 5.7 and MR = 5, so the total profit will decrease so it comes backwards
  • This 300 ⇒ MR = MC
  • Profit maximizing condition is Market Price should be equal to the marginal cost that is being spent on producing that unit of good
  • Condition is P = MC
  • The key idea here is that firms will produce as long as marginal revenue exceeds marginal cost.
  • Remember that a firm wants to maximize the difference between total revenue and total cost, not the difference between marginal revenue and marginal cost

Short Run Supply Curve

  • Marginal Cost Curve is the Supply Curve of the firm in the Short run
  • At any market price, the marginal cost curve shows the output level that maximizes the profit
  • The point below which there’s no production is the shut down point
