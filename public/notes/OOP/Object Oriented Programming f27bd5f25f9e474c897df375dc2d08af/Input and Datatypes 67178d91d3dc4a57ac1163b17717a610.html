Datatypes

  • Java is a strongly typed language
  • We have 8 primitive types: byte, short, int, long, char, float, double and Boolean
  • Integers:
  • To convert any number say 1000 to byte, we need to find the nearest 256 factor and use it and do 1000 - 256(4) = -24
  • Also float to long and float to double
  • When one type of data is assigned to another type of variable, an automatic type conversion will take place if the following two conditions are met:
    • The two types are compatible.
    • The destination type is larger than the source type.
  • When these two conditions are met, a widening conversion takes place. For example, the int type is always large enough to hold all valid byte values, so no explicit cast statement is required
  • This is only for automatic type conversion, casting can still narrow the size
  • First, all byte, short, and char values are promoted to int, as described. Then, if one operand is long, the whole expression is promoted to long. The entire expression is promoted to float if one operand is a float. If any of the operands are double, the result is double
  • By default floating point numbers are double

Arrays

  • We use
int month_days[] = new int[12];
// We can do these separately also
// Don't put number in the initial brackets
  • Arrays are initialized to the 0 value of that type by default, so we have 0 for int, false for Boolean, and null for other types
  • Arrays can also be initialized
int month_days[] = {1,2,3,4,5};
  • Multidimensional Arrays are used as:
int twoD[][] = new int[3][4];
  • We can always change the length of the 2nd dimension and have an array of arrays of variable length

Var

  • var can be used to implicitly infer datatype from an assigned value
  • It won’t work if there is no assigned value and will give us an error
var x = 10.0; // x is double
  • If we are using var with an array, we need to specify datatype on the right-hand side
