Question 5

  • Grammar = S→aBA; A→aA|c; B→bB | b
  • Input string abbac
  • In bottom up parsing with a stack after consuming ab, the stack status is #ab where # indicates bottom of stack and now the remaining part is bac, and the next input symbol is b. Now should we go with shifting next input symbol b on to stack or reducing b at the top of stack with B? Which is the right option?
  • Answer: Shift next input symbol b
